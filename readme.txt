Projeto exemplo para criação de WEB Services Rest.
Disponibiliza um serviço mockado de busca de endereço através do CEP.

Utilizado a especificação JAX-RS com a implementação Jersey para expor os serviços.

Utilização da biblioteca json para trabalhar com JSON

Utilizado JUnit e JMockit para testes unitários.

Futuramente adicionar uma implementação de DAO que permite utilize algum banco de dados ou até mesmo JPA.
